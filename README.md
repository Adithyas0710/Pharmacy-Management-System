Pharmacy Management System (MongoDB)
Overview

This project is a simple Pharmacy Management System developed using MongoDB. It helps manage billing, customer transactions, and medicine inventory efficiently through a basic web interface.
The system demonstrates CRUD operations and basic database interaction using a NoSQL database.

Features
Billing system with total calculation
Customer transaction search
Inventory management
Daily sales summary
Multiple payment modes

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Node.js
Database: MongoDB
Tools: VS Code, MongoDB Compass

How to Run the Project
Clone the repository
git clone https://github.com/your-username/pharmacy-management.git
Navigate to project folder
cd pharmacy-management
Install dependencies
npm install
Start the server
node server.js
Open browser and go to:
http://localhost:3000
